# Apostila: Segurança em Sistemas de Informação

Este repositório contém o material didático da disciplina de **Segurança em Sistemas de Informação** do curso de Bacharelado em Sistemas de Informação. O material está organizado em módulos, cobrindo desde os fundamentos até práticas avançadas de defesa e gestão.

## 📚 Estrutura do Material

O conteúdo está dividido em módulos temáticos, cada um contendo capítulos teóricos, exemplos práticos e listas de exercícios.

## 🛠️ Como utilizar este material

1.  **Navegação:** Você pode ler os capítulos diretamente no GitHub clicando nos links acima. O conteúdo está formatado em Markdown para facilitar a leitura em qualquer dispositivo.
2.  **Exercícios:** Ao final de cada módulo, utilize a lista de exercícios para fixação do conteúdo.
3.  **Contribuições:** Sugestões de melhorias, correções de erros de digitação ou novos exemplos práticos são bem-vindos via *Pull Requests*.
